AI-Based Software Defect Prediction System

Project Overview

This project uses Artificial Intelligence and Machine Learning to predict software defects before deployment. The system analyzes project size, complexity and team experience to identify potential risks and improve software quality.

Key Features

  • AI-Based Defect Prediction
  • Risk Score Calculation
  • Recommendations System
  • Prediction History
  • SQLite Database Integration
  • Analytics Dashboard
  • CSV Report Export

Technology Used

  • Python
  • Flask
  • Machine Learning (Scikit-Learn)
  • SQLite Database
  • HTML, CSS
